# Budget Tracker (Python)

A simple command-line budget tracker built using Python. This project allows users to track income and expenses and view their current balance.

## Features
- Add income
- Add expenses
- View total balance
- Simple menu-based interface

## Skills Used
- Python
- Functions
- Loops
- Conditionals
- User input and output

## How to Run
1. Download the file
2. Open terminal
3. Run:
   python budget_tracker.py

## Future Improvements
- Add categories (food, rent, etc.)
- Save data to a file
- Add error handling for invalid input
